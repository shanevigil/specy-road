# Agents (optional)

This directory is reserved for **thin**, editor-agnostic snippets if you want them (e.g. shared prompt fragments). Most teams rely on platform-native agent config (Cursor rules, Claude `CLAUDE.md`, etc.) and the repo entry point [`AGENTS.md`](../AGENTS.md) instead. For what the kit requires vs leaves optional, see [`../docs/philosophy-and-scope.md`](../docs/philosophy-and-scope.md).

Do not duplicate `constitution/`, `constraints/`, or `roadmap/` content here.
