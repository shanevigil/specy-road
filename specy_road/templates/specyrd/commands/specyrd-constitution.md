Thin stub — canonical behavior is `specy-road scaffold-constitution` and editing the repo files. Generated by specyrd {{SPECYRD_VERSION}}.

From the **repository root**:

```bash
cd "$(git rev-parse --show-toplevel)"
```

1. **Scaffold starter files** (skips files that already exist unless you pass `--force`):

```bash
specy-road scaffold-constitution
```

2. Edit **`constitution/purpose.md`** (why the effort exists) and **`constitution/principles.md`** (how you decide). These are **human judgment**, not machine-validated; enforceable rules live under `constraints/`.

3. Optionally use the **Gantt PM UI** (`specy-road gui`) → **Constitution** to edit and save the same paths in the browser.

4. Roadmap validation does not lint prose. After roadmap edits, run:

```bash
specy-road validate
specy-road export
```
