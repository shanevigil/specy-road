Thin stub — records implementation review in `roadmap/registry.yaml`. Generated by specyrd {{SPECYRD_VERSION}}.

Use when **`roadmap/git-workflow.yaml`** sets **`require_implementation_review_before_finish: true`**. You should have **`work/implementation-summary-<NODE_ID>.md`** first (see `work/implementation-summary.template.md`).

From the **feature branch** (`feature/rm-<codename>`), **repository root**:

```bash
cd "$(git rev-parse --show-toplevel)"
```

```bash
specy-road mark-implementation-reviewed
```

- Prints the summary file, then an interactive menu (walkthrough / approve / quit).
- Non-interactive: `specy-road mark-implementation-reviewed --yes`
- Emergency without summary file: `--allow-missing-summary` (warning).

Next: `specy-road finish-this-task`

See `docs/dev-workflow.md` (Finish).
