Thin stub — abort task pickup. Generated by specyrd {{SPECYRD_VERSION}}.

Use when you ran **`specy-road do-next-available-task`** (or claimed the same way) and want to **release the task** without completing it: the registry row is removed with a **new commit on the integration branch** and **`git push`** (same coordination contract as pickup). You must be on **`feature/rm-<codename>`** with a **clean** working tree.

From the **repository root**:

```bash
cd "$(git rev-parse --show-toplevel)"
```

```bash
specy-road abort-task-pickup
```

Optional: **`--base`**, **`--remote`** (same as other dev commands; defaults from **`roadmap/git-workflow.yaml`**). If your feature branch has **local commits** that are not on the remote integration branch, the command **refuses** unless you pass **`--force`** (deletes the local feature branch with **`git branch -D`**; can drop those commits locally). **`--force`** also removes **`work/implementation-summary-<NODE_ID>.md`** if present.

See **`specy-road abort-task-pickup --help`** and `docs/dev-workflow.md`.
